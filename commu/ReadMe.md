## 実行方法

実行方法

```cd src```

```make```

1. Server側起動

 ```java Server  ```

2. Client側起動

 ```./a.out  ```

# 概要

1. サーバは起動時に Task(リモートオブジェクト) のインスタンスを生成し，登録する. 
   1. **Taskクラスはstr1, str2文字数を半角英数5文字固定** (例ではstr1= Hello, str2 = World)
2. サーバが起動した後，クライアントはサーバで生成されたインスタンスを取得する. 
   1. **127.0.0.1::2434でシリアル通信**
3. クライアントは取得したインスタンス(シリアライズされたオブジェクト)を解析し，メンバ 変数 str1, str1 の値を取得する.そして，それら文字列を連結し，Person.name に代入したあ と，この Person オブジェクトを引数にしてサーバの hello を呼び出す. 
   1. **クライアントではstr1, str2が10文字固定にされているのを前提にパースして取得**
   2. **バイナリとパース結果をコンソールに表示**
   3. **127.0.0.1::2434でシリアル通信**
4. サーバでは引数 (Person) を受け取り，デシリアライズしたオブジェクトを使って hello を呼 び出す.その結果，サーバのコンソールに str1+str2 の結果が表示される. 
   1. **受け取ったPersonをバイナリでコンソールに表示**
