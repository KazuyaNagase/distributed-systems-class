## 実行方法

0. client01フォルダに移動

   ```cd client01```

1. ライブラリ依存関係の解決

   ```npm install```

2. TypeScriptをコンパイルして単一ファイルにまとめる

   ```npm run build```

3. 作成したファイルを実行

   ```node ./dist/main.js ```

4. client02フォルダに移動

   ```cd client02```

5. ライブラリ依存関係の解決

   ```npm install```

6. TypeScriptをコンパイルして単一ファイルにまとめる

   ```npm run build```

7. 作成したファイルを実行

   ```node ./dist/main.js ```

8. それぞれの初期タイムスタンプ時刻をキーボードから入力する

   ```12 (例)```
